# Copy configs

```
git clone -b https://github.com/d-solis/dotfiles/tree/hyprlandv2
cp -r dotfiles/config/eww ~/.config/eww
cp -r dotfiles/config/hypr ~/.config/hypr
mv ~/.config/hypr/_hyprland.conf ~/.config/hypr/hyprland.conf
```
